Installation Instructions

Windows

  1. Open the native-host folder in File Explorer
  2. Double-click install_host.bat
  3. Follow the on-screen prompts to complete installation

macOS / Linux

  1. Open a terminal in the native-host folder
  2. Run: chmod +x install_host.sh && ./install_host.sh 
    chmod +x install_host.sh && ./install_host.sh
  3. Follow the on-screen prompts to complete installation

After installation, click "Check Connection" to verify.
